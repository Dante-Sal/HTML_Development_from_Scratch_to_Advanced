This is a form (uses the <form> tag):

The "value" attribute is the default value and the value that is sent to the server. Some input types require a specific format in this attribute to function correctly.

The "checkbox" type input sends the value of the "value" attribute to the server only if it is checked (otherwise, it sends nothing). If the "value" attribute is not specified, the default value is "on".

The "file" type input sends the path of the selected file to the server only if the user selects something (otherwise, it sends an empty string).

If the "src" and "alt" attributes are not specified in an "image" type input, the value of the "value" attribute will be the text displayed next to the default image icon.

The "radio" type input sends the value of the "value" attribute of the selected input to the server only if any input from the group is selected (otherwise, it sends nothing).
