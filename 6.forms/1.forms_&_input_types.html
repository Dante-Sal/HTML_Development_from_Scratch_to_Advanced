This is a form (uses the <form> tag):

The "button" type input displays a clickable button.

The "checkbox" type input displays a checkable box.

The "color" type input displays a color selector.

The "date" type input displays a date-only selector.

The "datetime-local" type input displays a date & time selector.

The "email" type input displays a text box that only accepts email format when a "submit" type input is activated.

The "file" type input displays a file selector.

The "hidden" type input does not display anything and is used to send data that should not be seen or modified by users.

The "image" type input displays an image "submit" button where the image is specified in the "src" attribute. It usually also has an alternative text.

The "month" type input displays a selector where the user can choose a month from a specific year.

The "number" type input displays a text box that only accepts numbers when a "submit" type input is activated.

The "password" type input displays a text box that hides its contents using a mask.

The "radio" type input is usually used in groups, and in each group the user can only select one option from several.

The "range" type input displays a slider to select a specific number between two limits.

The "reset" type input displays a button to reset all form input values to their default values.

The "search" type input displays a text box that has a button to delete the text entered.

The "submit" type input displays a button to submit the entered form data.

The "tel" type input displays a text box that only accepts telephone number format when a "submit" type input is activated.

The input without a "type" attribute or of type "text" displays a text box.

The "time" type input displays a time-only selector.

The "url" type input displays a text box that only accepts URL format when a "submit" type input is activated.

The "week" type input displays a selector where the user can choose a week from a specific year.
